# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the Ryumem project.

ADRs are documents that capture important architectural decisions made along with their context and consequences.

## ADR Template

When creating a new ADR, use the following template:

```markdown
# ADR-XXX: Title

## Status
Proposed | Accepted | Deprecated | Superseded

## Context
What is the issue that we're seeing that is motivating this decision?

## Decision
What is the change that we're proposing and/or doing?

## Rationale
Why is this change being made? What alternatives were considered?

## Consequences
What becomes easier or more difficult because of this change?
```
